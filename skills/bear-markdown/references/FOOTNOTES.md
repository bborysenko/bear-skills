# Footnotes Reference

## Syntax

```markdown
Text with a footnote reference[^1].

More text with another reference[^2].

[^1]: This is the first footnote.
[^2]: This is the second footnote.
```

## Rules

- Footnotes are auto-numbered starting at 1
- Footnote definitions go at the bottom of the note
- Click or tap a footnote number to jump between the reference and its definition
- Keyboard shortcut to insert: `Cmd+Shift+E` (Mac)

## Bear-Specific Behavior

- Bear manages footnote numbering automatically (sequential from top of note)
- Footnote references and definitions are linked for bidirectional navigation
- The footnote UI is the primary way to create footnotes (via the three-dot menu in the formatting bar or keyboard shortcut)
